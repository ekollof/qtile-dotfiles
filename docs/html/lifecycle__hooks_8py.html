Detailed Description

Consolidated lifecycle hooks for qtile.

This module consolidates all lifecycle-related hooks into a single, well-organized system with improved maintainability and error handling.

Features
  • Screen change detection and handling
  • Startup and shutdown hooks
  • Monitor hotplug support
  • Window retiling after restart
  • Comprehensive error handling and validation
